# ChatGPT Exporter File Extraction Roadmap

This folder contains the implementation roadmap and prompt packs for extending ChatGPT Exporter into a file extraction, asset recovery and personal archive platform.

| Phase | Document |

## Delivery Structure

- Phase 1: File discovery, extraction and ZIP export.
- Phase 2: Advanced recovery, catalogue, incremental export and automation.
- Phase 3: Personal archive and knowledge platform.

## Operating Principle

Implement in small, buildable increments. Each chunk must compile before moving to the next.
